# EVALUACION 1 - Programación II
# GESTION DE PEDIDOS EN UN RESTAURANT

Este proyecto corresponde a la **Evaluación 1 de Programación II**.  
Es una aplicación Android desarrollada en **Kotlin** que simula la gestión de pedidos en un restaurante.

---

## Repositorio
El código fuente está disponible en GitHub:  
[Programacion-II-Ev1](https://github.com/marc7soto-web/Programacion-II-Ev1)

---

## Funcionalidades principales
- Cálculo automático de **subtotal, propina y total**.
- Interfaz gráfica implementada en MainActivity.
- Modelos de datos:
    - ItemMenu
    - CuentaMesa
- Recursos centralizados en:
    - res/values/strings.xml (textos)
    - res/drawable (imágenes)
